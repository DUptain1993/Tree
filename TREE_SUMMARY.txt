PARTITION LAYOUT SUMMARY:
-------------------------
- Boot Partitions: boot, init_boot, vendor_boot, dtbo (all A/B)
- Dynamic Super Partition: 8 GB (contains system, vendor, product, etc.)
- User Data: ~110 GB (F2FS, encrypted)
- Firmware: Multiple A/B partitions for seamless updates
- Modem: md1img (A/B slots)
- Connectivity: Separate WiFi, BT, GNSS firmware partitions

IMPORTANT NOTES:
----------------
1. This device uses A/B partitioning - both slots must be considered
2. Dynamic partitions require special handling in recovery
3. Encryption keys are stored in metadata partition
4. MediaTek platform requires specific HALs and drivers
5. Trustonic TEE is used for security features
